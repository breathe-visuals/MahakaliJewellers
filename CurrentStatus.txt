# Mahakali Jewellers Live Web App Status

## Included
- Express + Socket.IO backend
- Hidden source layer
- Gold rate calculator (24K to 9K, no 23K)
- Silver product feed
- Coin feed
- Future and spot sections
- Service worker shell cache
- Media folder for favicons and logo assets

## Deployment
Use Render Web Service.

## Folders
- `Media/` for favicon assets
- Root files:
  - `index.html`
  - `styles.css`
  - `app.js`
  - `server.js`
  - `sw.js`
